# Smart WiFi Performance Analyzer

## Overview

Smart WiFi Performance Analyzer is a Python application that scans nearby WiFi networks, analyzes signal strength, identifies channel congestion, and recommends the best WiFi channel.

## Features

- WiFi Network Scanning
- Signal Strength Analysis
- Strongest Network Detection
- Channel Recommendation
- Channel Congestion Visualization
- CSV Report Download
- Interactive Streamlit Dashboard

## Technologies Used

- Python
- Streamlit
- Pandas
- PyWiFi
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```

## Example Output

- Strongest Network Detection
- Recommended WiFi Channel
- Signal Strength Graph
- Channel Congestion Graph
